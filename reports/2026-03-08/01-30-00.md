### 2. [Show HN: Argus – Claude Code 会话调试器](https://news.ycombinator.com/item?id=47288571)

#### 核心内容
- Argus 是一款 VS Code 扩展，以希腊神话中的"全视巨人"命名，专门用于调试和分析 Claude Code 会话

#### 关键讨论
- 帮助开发者识别浪费的 API 调用、优化 token 使用、检测重试循环和重复操作
- 提供详细的成本分析（每步 token 可视化、缓存命中率、模型归属）和性能指标
- 使用 React 19 + D3.js 构建交互式可视化，支持实时会话监控
- 开源项目，TypeScript 编写，MIT 许可

#### 分析价值
⭐⭐⭐⭐ 填补了 Claude Code 开发工作流的工具空白，帮助开发者理解和优化 AI 辅助编程模式，降低使用成本
