**热度:** 318 points | **评论:** 416 条  
**来源:** [TechCrunch](https://techcrunch.com/2026/02/27/openai-raises-110b-in-one-of-the-largest-private-funding-rounds-in-history/)

#### 核心内容
- OpenAI 宣布完成 **1100 亿美元** 私人融资，创历史最大私人融资轮次纪录
- 投资方构成：
  - **Amazon:** 500 亿美元（其中 350 亿美元与 AGI 实现或 IPO 挂钩）
  - **Nvidia:** 300 亿美元
  - **SoftBank:** 300 亿美元
- 投前估值：**7300 亿美元**（相比 2025 年 3 月的 3000 亿美元翻倍有余）
- 基础设施合作：
  - 与 Amazon Bedrock 合作开发"状态化运行时环境"
  - 承诺消耗至少 2GW AWS Trainium 算力
  - 与 Nvidia 承诺使用 3GW 专用推理容量和 2GW Vera Rubin 训练系统

#### 价值点
1. **行业信号:** 巨额融资表明 AI 基础设施军备竞赛进入白热化阶段，资本对 AGI 前景极度乐观
2. **估值泡沫讨论:** 7300 亿估值引发社区对 AI 泡沫的担忧，类似 2000 年互联网泡沫
3. **算力绑定:** 大部分投资以算力服务形式而非现金，反映 AI 公司核心瓶颈是算力而非资金
4. **AGI 对赌:** Amazon 的 350 亿美元与 AGI/IPO 挂钩，显示产业界对 AGI 时间表有明确预期

#### 社区讨论焦点

### 3. [Anthropic 为开源维护者免费提供 Claude Max](https://news.ycombinator.com/item?id=47178371)

**热度:** 393 points | **评论:** 183 条  
**来源:** [Anthropic 官方](https://claude.com/contact-sales/claude-for-oss)

#### 核心内容
- Anthropic 推出"Claude for Open Source"计划
- **福利:** 符合条件的开源维护者可获得 **6 个月免费 Claude Max 20x**
- **申请资格:**
  - 主要维护者或核心团队成员
  - 项目拥有 5000+ GitHub stars 或 100 万 + 月 NPM 下载量
  - 过去 3 个月内有提交、发布或 PR 审核记录
- **名额限制:** 最多接受 10,000 名贡献者
- 申请滚动审核

#### 价值点
1. **生态建设:** Anthropic 通过支持开源社区建立开发者忠诚度，与 GitHub Copilot 竞争
2. **实际价值:** Claude Max 20x 市场价值约$400/月，6 个月相当于$2400 福利
3. **开源依赖:** 承认开源生态对 AI 行业的基础性作用
4. **人才争夺:** 通过免费服务吸引潜在用户，培养使用习惯

#### 社区讨论焦点
- 申请门槛是否过高（小项目维护者被排除）
- 与 GitHub Copilot 免费计划的对比
- 是否存在数据使用条款陷阱
